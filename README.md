# Классификация научных статей arXiv

## Задача

Многоклассовая классификация научных статей по 8 верхнеуровневым категориям таксономии arXiv (cs, econ, eess, math, physics, q-bio, q-fin, stat) на основе заголовка и аннотации статьи.

## Данные

Использовался публичный датасет arXiv от Корнеллского университета с Kaggle ([Cornell-University/arxiv](https://www.kaggle.com/datasets/Cornell-University/arxiv)), содержащий метаданные более миллиона статей (~5 ГБ). Из него была сформирована сбалансированная выборка: по 25 000 статей на каждый из 8 классов, итого 200 000 примеров. Входом модели служит конкатенация заголовка и аннотации.

## Модели и эксперименты

Было обучено три модели на базе HuggingFace Transformers:

| Модель | Описание | Accuracy / F1 |
|---|---|---|
| DistilBERT (hard labels) | Стандартное дообучение с one-hot метками | ~0.85 |
| DistilBERT (soft labels) | Обучение с размягчёнными метками как форма регуляризации | ~0.85 |
| SciBERT | Модель предобученная на корпусе научных статей | ~0.90 |

Лучший результат показал SciBERT благодаря доменно-специфичному предобучению на научных текстах, что естественно соответствует задаче.

## Наблюдения

Наиболее сложной оказалась граница между cs и stat: сообщества cs.LG и stat.ML на arXiv реально пересекаются, и модель отражает эту объективную неоднозначность. Остальные классы разделяются уверенно за счёт специфической терминологии каждой области.
