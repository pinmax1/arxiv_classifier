# Классификация научных статей arXiv

## Задача

Многоклассовая классификация научных статей по 8 верхнеуровневым категориям таксономии arXiv (cs, econ, eess, math, physics, q-bio, q-fin, stat) на основе заголовка и аннотации статьи.

## Данные

Использовался публичный датасет arXiv от Корнеллского университета с Kaggle ([Cornell-University/arxiv](https://www.kaggle.com/datasets/Cornell-University/arxiv)), содержащий метаданные более миллиона статей (~5 ГБ). Из него была сформирована сбалансированная выборка: по 25 000 статей на каждый из 8 классов, итого ~175000 примеров, так как не всех классов есть по 25000. Входом модели служит конкатенация заголовка и аннотации.

## Модели и эксперименты

Было обучено три модели на базе HuggingFace Transformers:

| Модель | Accuracy / F1 |
|---|---|
| DistilBERT (hard labels) | ~0.86 |
| DistilBERT (soft labels) | ~0.84 |
| SciBERT | ~0.88 |

Лучший результат показал SciBERT благодаря доменно-специфичному предобучению на научных текстах, что естественно соответствует задаче.

## Наблюдения

Наиболее сложной оказалась граница между cs и stat: сообщества cs.LG и stat.ML на arXiv реально пересекаются, и модель отражает эту объективную неоднозначность. Остальные классы разделяются уверенно моделью.
